快速开始

三种方式，总有一种适合你

💻 桌面开发

git clone https://github.com/coder-kingyifan/king-remind.git
cd king-remind
npm install
npm run dev

🐳 Docker 部署

git clone https://github.com/coder-kingyifan/king-remind.git
cd king-remind
docker compose run --rm king-remind

🔧 Windows 打包

npm run build
npm run pack

# 产物:
# king-remind-{ver}-setup.exe
# king-remind-{ver}-portable.exe
